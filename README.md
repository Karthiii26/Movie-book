# Movie Ticket Booking System (MERN Stack)

A fully functional movie ticket booking platform built for college evaluation. It features a modern dark UI, JWT authentication, and a complete booking lifecycle.

## Tech Stack
- **Frontend**: React, React Router, Context API, Lucide React
- **Backend**: Node.js, Express, JWT, Bcrypt
- **Database**: MongoDB (Mongoose)
- **Styling**: Vanilla CSS (Rich Aesthetic)

## Features
- **Auth**: Register and Login with secured JWT tokens.
- **Movies**: Browse active movies with detailed info.
- **Theatres**: View theatres and available showtimes.
- **Seats**: Visual seat selection with real-time availability.
- **Bookings**: Confirm bookings and view history in Dashboard.

## How to Run Locally

### Prerequisites
- Node.js installed
- MongoDB running locally (default: `mongodb://localhost:27017/movie-booking`)

### Step 1: Backend Setup
1. Open the `backend` folder.
2. Run `npm install`.
3. Create a `.env` file (one is already provided in this project).
4. Run `node seed.js` to populate the database with mock data.
5. Run `node server.js` to start the backend server on port 5000.

### Step 2: Frontend Setup
1. Open the `frontend` folder.
2. Run `npm install`.
3. Run `npm run start` to start the frontend on port 3000.
